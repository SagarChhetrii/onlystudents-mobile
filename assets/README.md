# App Assets

This directory should contain the following image files:

- **icon.png** (1024×1024) - App icon
- **splash.png** (1242×2436) - Launch splash screen
- **adaptive-icon.png** (1080×1080) - Android adaptive icon
- **favicon.png** (192×192) - Web favicon

These are referenced in `app.json` and are required for building/deploying the app.

For development in Expo Go, the app runs without these assets.
